# Headless HTML Streamer 🚀

Этот проект позволяет запускать любое локальное веб-приложение (HTML/JS) в изолированном headless-браузере и транслировать его напрямую на YouTube (или Twitch) в режиме реального времени. 

Проект полностью упакован в Docker, не требует установки графического окружения на хост-машину и адаптирован для вертикальных трансляций (Shorts, Reels, TikTok) в разрешении **1080x1920**.

## 🛠 Технологический стек
* **Docker** — для изоляции окружения.
* **Xvfb** — виртуальный дисплей для рендеринга графики без физического монитора.
* **Puppeteer (Chromium)** — запуск веб-страницы в режиме Kiosk (без вкладок, плашек и интерфейса).
* **FFmpeg** — захват виртуального экрана и стриминг по протоколу RTMP с генерацией пустой аудиодорожки (обязательное требование YouTube).

## 📂 Структура проекта
* `index.html` — само веб-приложение (в данном примере: миллисекундный неоновый таймер).
* `browser.js` — Node.js скрипт, который открывает HTML-файл в Chromium с нужными флагами.
* `stream.sh` — главный bash-скрипт. Поднимает Xvfb, запускает браузер, стартует FFmpeg и обрабатывает корректное завершение по `Ctrl+C`.
* `Dockerfile` — инструкции для сборки легковесного Linux-образа со всеми нужными зависимостями.

## 🚀 Установка и запуск

### 1. Подготовка
Убедитесь, что на вашем компьютере установлен и запущен **Docker** (или OrbStack для macOS).

### 2. Сборка образа
Откройте терминал в папке с проектом и выполните сборку (это займет пару минут при первом запуске):

```bash
docker build -t headless-timer .
```

### 3. Запуск трансляции
Получите ваш **Ключ трансляции (Stream Key)** в панели управления прямыми трансляциями YouTube. Запустите контейнер, передав ключ через переменную окружения:

```bash
docker run -it --rm \
  -e STREAM_KEY="ВАШ_КЛЮЧ_ТРАНСЛЯЦИИ" \
  -e CHANNEL_ID="ВАШ_ID_КАНАЛА_НАЧИНАЕТСЯ_С_UC" \
  headless-timer
```
* **Внимание:** Флаг `-it` обязателен! Он позволяет пробрасывать сигналы терминала внутрь контейнера, чтобы вы могли корректно остановить стрим.

### 4. Остановка трансляции
Для завершения стрима просто нажмите `Ctrl + C` в терминале. Скрипт перехватит сигнал, безопасно завершит процессы FFmpeg и браузера, после чего контейнер автоматически удалится (благодаря флагу `--rm`).

## ⚙️ Кастомизация

* **Изменить контент:** Просто отредактируйте `index.html`. Добавляйте CSS-анимации, графики или виджеты. Браузер отрисует всё, что работает в обычном Chrome. 
* **Язык страницы:** В `index.html` установлен `lang="en"`. Не меняйте его на русский, иначе Chromium может попытаться вызвать панель Google Translate, которая попадет в кадр.
* **Сменить платформу:** Если вы хотите стримить на Twitch или другой сервис, измените RTMP-ссылку в последней строке файла `stream.sh`:
  * YouTube: `rtmp://a.rtmp.youtube.com/live2/$STREAM_KEY`
  * Twitch: `rtmp://live.twitch.tv/app/$STREAM_KEY`
